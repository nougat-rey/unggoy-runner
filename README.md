# unggoy-runner
Play as an unggoy (grunt) from the Halo universe as it tries to evade the flood infection form coming after it! Try to survive as long as possible!

## Run Instructions

```
cd code
python main.py
```

## Game Instructions

+ Press Space to jump
+ Press Return to return to main menu
